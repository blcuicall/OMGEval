Um modelo estatístico é um modelo matemático que incorpora um conjunto de suposições estatísticas sobre a geração de dados de amostra (e dados similares de uma população maior). Um modelo estatístico representa, muitas vezes de maneira bastante idealizada, o processo gerador de dados. Um modelo estatístico geralmente é especificado como uma relação matemática entre uma ou mais variáveis aleatórias e outras variáveis não aleatórias. Nesse sentido, um modelo estatístico é "uma representação formal de uma teoria" (Herman Adèr citando Kenneth Bollen). Todos os testes de hipóteses estatísticas e todos os estimadores estatísticos são derivados através de modelos estatísticos.
